Adds a simple HTML widget with syntax highlighting for HTML, CSS and JS.

== Description ==
HTML Widget adds a new widget that can be used for displaying any custom HTML in a easy to use way. The syntax highlighting makes the code clear and readable which makes your widgets easier to manage. Only users with sufficient capabilities can use all HTML elements, while users with limited capabilities can only use safe HTML that is allowed in posts.

== Installation ==
1. In your admin panel, navigate to the Plugins screen.
2. Type in HTML Widget in the search form and press the \'Enter\' key on your keyboard.
3. Click on the \'Activate\' button to use your new plugin right away.
4. Navigate to Appearance > Widgets or Appearance > Customize > Widgets.
5. Add as many HTML Widgets as you like and customize to taste.
